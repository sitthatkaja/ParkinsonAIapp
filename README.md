# Parkinson App for collected data with Expo Go
---

For the video recording application, the researcher developed code to be used together with React Native via Expo Go. To run the application, it must be launched through a working terminal, for which the researcher recommends using Visual Studio Code, and the QR code must be scanned via Expo Go. Therefore, for convenience, the researcher recommends downloading Visual Studio Code for code execution and Expo Go for recording videos through the mobile application.
The code used by the researcher has been uploaded to the GitHub platform and can be accessed at https://github.com/sitthatkaja/ParkinsonAIapp. The usage process is described as follows:
1.	Go to https://github.com/sitthatkaja/ParkinsonAIapp
2.	Click Code, then select Download ZIP.
3.	Extract the ZIP file.
4.	Open Visual Studio Code.
5.	Select Open Folder and choose the extracted folder.
6.	Open the Terminal and run the command: ```npx expo start```
7.	After that, a QR code will be generated for use.
8.	Scan the QR code using Expo Go, after which the application can be used.

***Note: The command versions in Visual Studio Code must be updated to the latest version, which can be observed from the yellow text shown on the QR code as follows. 
The command ```npm install -g expo-cli``` can be used as a basic command for updating.
When using the video recording application according to the researcher’s setup, upon entering the application interface, the Start button will appear. Press it and follow the video recording process. After completion, the recorded video will be saved to the phone’s video gallery.***

***Note: The entire process can be customized through the code in Visual Studio Code, depending on the user.***
